# Laravel Blog "Lablog"

## Just like personal Blog

## Progress

- Laravel 5.2.*
- Setup Frontend
- Setup Backend
- Setup database


## How to clone

- git clone https://github.com/astadipati/lablog.git
- Create database and place the same name at .env file in laravel project folder
- composer install  
- change APP_KEY=blablabla
- php artisan key:generate
- php artisan migrate
